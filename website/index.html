⚡

Instant Capture

Global hotkey freezes your screen. Select any region, snap to windows, or capture full screen. Multi-monitor support with cross-screen stitching.

✏️

Click-to-Edit Annotations

Click any annotation to select it, then edit stroke, style, color, and fill in real time. Full undo/redo. Rotate, resize, and reposition without switching tools.

📸

Scroll Capture

Select a region and scroll. macshot stitches it into one tall image with Apple Vision. Live preview updates as you scroll.

🎬

Screen Recording

Record any area as MP4 (up to 120fps) or GIF. System audio, microphone, mouse highlights. Built-in trim editor.

👁‍🗨️

Smart Censor

Pixelate, blur, solid fill, or erase. Auto-redact PII (emails, phones, API keys), detect faces and people. Erase mode intelligently matches surrounding colors.

✨

Beautify

30 gradient backgrounds including mesh gradients. macOS window frame with traffic lights, shadow, adjustable padding and corner radius.

🔍

OCR & Translate

Extract text with Apple Vision. Auto-detects all languages. Copy to clipboard, translate to 30+ languages, or search with Google AI.

☁️

Upload Anywhere

One-click upload to Google Drive, imgbb, or any S3-compatible service (Cloudflare R2, AWS, MinIO). Link copied instantly.

⚡

Lightweight & Native

~8 MB memory at idle. Pure Swift + AppKit. No Electron, no web views, no bloat. Lives quietly in your menu bar.
